## Overview
This project is an advanced **food recommendation and conversational AI system** built using **Chroma DB** and Python. It demonstrates three distinct approaches to similarity search and AI-driven interaction using a rich food dataset that includes nutritional data, ingredients, cooking methods, and taste profiles.

The system showcases real-world applications of **vector databases**, **similarity search**, and **Retrieval-Augmented Generation (RAG)** through fully interactive command-line interfaces (CLI).
